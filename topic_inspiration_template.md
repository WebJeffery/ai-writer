# 公众号爆款选题灵感表模板

此模板基于“情绪驱动 + 价值稀缺 + 社交货币”的底层逻辑设计，旨在帮助创作者系统化地捕捉灵感、打磨选题。

## 选题方法论速查

- **底层逻辑**: 情绪 (共鸣) + 价值 (稀缺信息) + 社交 (展示谈资)
- **构思技巧**:
    - **反向思考**: 质疑常识 (e.g., 加班 != 努力)
    - **痛点放大**: 具体场景 + 数据 (e.g., 房租占工资50%)
    - **故事包装**: 冲突 + 代入感
    - **清单/攻略**: 实用 + 避坑
    - **对比反差**: 强烈对照 (e.g., 穷人思维 vs 富人思维)
