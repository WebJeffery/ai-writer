# AI 写作私有知识库使用指南

## 简介
这是您的核心资产库。通过整理您的文风样本、个人故事和领域知识，您可以让 AI (ChatGPT, Claude, Kimi 等) 真正成为您的“数字分身”。

## 目录结构说明

### 1. 文风样本 (`/文风样本`)
**存放内容**：您写过的、数据最好的文章（建议 5-10 篇）。
**用途**：让 AI 学习您的句长、语气、常用词汇。
**建议格式**：
- `我的代表作合集.txt` (推荐合并成一个大文件)
- `文章_标题.md`

### 2. 故事库 (`/故事库`)
**存放内容**：您的个人经历、创业故事、感悟。
**用途**：文章中插入真实的个人故事，增加“人味”和信任感。
**建议格式**：
- `【标签】故事标题.md`
- 例如：`【创业】第一次融资失败.md`

### 3. 领域知识 (`/领域知识`)
**存放内容**：行业黑话、独特的观点体系、数据报告、SOP。
**用途**：确保 AI 输出的内容专业、深刻。
**建议格式**：
- `行业术语表.md`
- `核心观点库.md`

## 如何使用
1. **直接投喂**：将相关文件内容复制或上传到 AI 对话框（Kimi/Claude 支持长文本）。
2. **构建 GPTs/Bot**：将这些文件上传到 ChatGPT GPTs 或 Coze 的知识库中。
3. **本地调用**：使用本项目提供的脚本（开发中）来检索和组合提示词。
