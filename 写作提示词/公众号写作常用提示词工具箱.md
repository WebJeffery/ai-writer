# 公众号写作常用提示词工具箱

这是一个模块化的公众号写作提示词库，你可以根据需要直接复制对应的 Prompt 使用。

---

## 1. 🎯 选题与灵感篇

### 1.1 爆款选题挖掘
> 适用于：不知道写什么，想结合热点或痛点找灵感。

```markdown
我是一个[领域，如：职场/育儿/AI]领域的公众号博主，我的目标用户是[描述用户，如：25-30岁的城市白领]。
请结合当前的社会热点或该人群的痛点，为我推荐5个有爆款潜质的选题。
要求：
1. 选题要具体，不要太宽泛。
2. 每个选题提供3个切入角度（如：反常识、情感共鸣、干货教程）。
3. 预测每个选题的用户情绪点（焦虑、好奇、爽感）。
```

### 1.2 竞品爆文拆解
> 适用于：看到一篇好文章，想分析它为什么火，并学习其逻辑。

```markdown
请分析这篇公众号文章（粘贴文章内容或链接）：
1. **核心痛点**：它解决了用户的什么问题或满足了什么心理？
2. **结构拆解**：分析它的行文逻辑和段落安排。
3. **爆点分析**：找出文中引起读者强烈共鸣的3个金句或段落。
4. **模仿建议**：如果我要写一篇同类主题的文章，如何写出新意？
```

## 2. 🏆 标题优化篇

### 2.1 爆款标题生成器 (多风格)
> 适用于：文章写好了，需要几个好标题来测试。

```markdown
请为主题为“[文章主题]”的文章，生成10个公众号标题。
请包含以下几种风格：
1. **悬念好奇型**：设置悬念，引发点击欲望。
2. **痛点扎心型**：直接戳中用户焦虑或痛点。
3. **利益承诺型**：明确告诉用户看完能得到什么。
4. **反差/反常识型**：打破常规认知。
5. **权威背书型**：利用数据或名人效应。

请为每个标题打分（0-100）并说明理由。
```

### 2.2 标题诊断与优化
> 适用于：已有标题，想让它更吸引人。

```markdown
我目前的标题是：“[原标题]”。
请扮演一位拥有百万粉丝的公众号主编，直言不讳地指出这个标题的问题（如：太平淡、太长、没重点）。
然后，请给出5个优化后的版本，并解释为什么新版本更好。
```

## 3. ✍️ 内容创作篇

### 3.1 黄金开头生成 (3秒法则)
> 适用于：开头写不出，或者开头太无聊留不住人。

```markdown
我正在写一篇关于“[文章主题]”的文章。
请帮我写3个不同版本的开头，要求在3秒内抓住读者注意力：
版本1：**场景代入法**（描述一个具体的、扎心的场景）。
版本2：**提问直击法**（连续发问，戳中痛点）。
版本3：**金句/数据法**（用震撼的数据或反常识的结论开场）。
每个开头控制在100字以内。
```

### 3.2 故事润色与画面感增强
> 适用于：文字太干巴，缺乏感染力。

```markdown
请润色下面这段文字，使其更具画面感和感染力。
运用“Show, Don't Tell”（展示而非讲述）的原则，多描写动作、神态、环境细节，少用形容词。
同时，适当加入心理描写，让读者能感同身受。

[粘贴原段落]
```

### 3.3 金句/语录生成
> 适用于：文章缺乏传播点，需要一些可以直接发朋友圈的句子。

```markdown
基于主题“[文章主题]”或以下内容：
[粘贴文章大纲或部分内容]

请生成5-8条“金句”。
要求：
1. 短小精悍，朗朗上口。
2. 具有情绪煽动性或哲理性。
3. 适合读者直接复制发朋友圈。
例如：“不要假装努力，结果不会陪你演戏。”
```

## 4. 🎨 结构与排版篇

### 4.1 全文大纲生成 (结构化)
> 适用于：有想法但没逻辑，需要梳理文章骨架。

```markdown
请为主题“[文章主题]”设计一个标准的公众号文章大纲。
请根据文章类型选择最合适的结构：
1. **SCQA模型** (情境-冲突-问题-答案)：适合干货、科普类。
2. **PREP模型** (观点-理由-案例-观点)：适合观点评论类。
3. **总分总结构**：适合盘点、清单类。

大纲应包含：
1. 引人入胜的开头。
2. 3-4个核心小标题（小标题要吸引人，不要像教科书）。
3. 每个部分的论述重点和素材建议（案例/数据）。
4. 强有力的结尾和行动号召 (CTA)。
```

### 4.2 排版视觉建议
> 适用于：不知道如何配图，或不知道怎么分段好看。

```markdown
我写了一篇关于“[文章主题]”的文章。
请给出视觉排版建议：
1. **配图建议**：需要什么样的封面图？正文中哪些地方需要插图？描述画面内容。
2. **重点标记**：哪些金句或关键段落适合加粗或标红？
3. **分段建议**：如何分段能适应手机屏幕阅读（如：每段不超过3-4行）。
```

## 5. 🤝 运营与互动篇

### 5.1 朋友圈推广文案
> 适用于：文章发了，需要在朋友圈吆喝一声。

```markdown
我的公众号新发了一篇文章，标题是《[文章标题]》。
核心内容是：[简述内容]。
请帮我写3条朋友圈推广文案：
1. **走心感悟版**：以个人口吻分享写作初衷。
2. **干货提炼版**：列出文章的3个核心知识点。
3. **互动提问版**：抛出一个问题，引导大家去文章里找答案。
```

### 5.2 评论区神回复
> 适用于：不知道怎么回复读者的评论，或者想引导更多互动。

```markdown
读者的评论是：“[粘贴评论内容]”。
请帮我拟定3个回复，要求：
1. 幽默风趣，或者温暖走心（根据评论基调）。
2. 能引发对方或他人的二次回复。
3. 如果是杠精评论，请给出高情商、不卑不亢的回击。
```

## 6. 🤖 风格化写作 (高级)

### 6.1 风格模仿 (仿写)
> 适用于：想模仿某个大V的风格（如：罗辑思维、咪蒙、半佛仙人等）。

```markdown
请分析这段文字的语言风格（语气、用词、句式长度、修辞手法）：
“[粘贴目标风格的样文]”

然后，请用同样的风格，以“[你的主题]”为题，写一段200字左右的开头。
```

## 7. 📚 经典写作公式库 (Templates)

### 7.1 PAS 模型 (痛点-煽动-解决)
> 适用于：直击痛点的干货文或种草文，转化率极高。

```markdown
请使用 **PAS (Problem-Agitation-Solution)** 公式为主题“[文章主题]”写一篇文章大纲或开头。
1. **Problem (提出问题)**：描述用户当前面临的困扰，越具体越好。
2. **Agitation (煽动痛苦)**：描述如果这个问题不解决，会带来什么严重后果？（制造焦虑/紧迫感）。
3. **Solution (提出方案)**：引出你的解决方案（文章核心内容/产品），并说明它如何完美解决上述问题。
```

### 7.2 AIDA 模型 (注意-兴趣-欲望-行动)
> 适用于：软文、推广文，引导用户一步步入坑。

```markdown
请使用 **AIDA** 模型撰写一篇推广文案：
1. **Attention (引起注意)**：用一个炸裂的标题或开头吸引眼球。
2. **Interest (激发兴趣)**：讲一个有趣的故事或展示惊人的数据，让读者想继续读下去。
3. **Desire (勾起欲望)**：展示产品/方法带来的美好改变（Before & After），让读者渴望拥有。
4. **Action (促成行动)**：给出明确的行动指令（如“点击链接”、“关注我”、“领取资料”）。
```

### 7.3 英雄之旅 (Hero's Journey) 简化版
> 适用于：个人成长故事、品牌创业故事。

```markdown
请用“英雄之旅”模型帮我梳理一个关于“[故事主题]”的故事脉络：
1. **平凡世界**：主角原本的平庸/困顿生活状态。
2. **召唤与拒绝**：遇到了什么契机？一开始是不是犹豫了？
3. **试炼与盟友**：经历了什么困难？谁帮助了你？
4. **至暗时刻**：遇到的最大危机是什么？
5. **通过试炼**：如何战胜危机？获得了什么顿悟？
6. **满载而归**：现在的状态如何？给读者什么启示？
```

## 8. 🔥 爆款元素增强 (Booster)

### 8.1 情绪钩子 (Emotional Hook)
> 适用于：文章太平淡，需要增加情绪张力。

```markdown
请检查这篇文章（或大纲），并在关键位置植入“情绪钩子”：
1. **开头**：加入好奇或愤怒的情绪，让读者欲罢不能。
2. **文中**：找到2个位置，通过“反差”或“共鸣”引发读者的情绪波动（如：从绝望到希望）。
3. **结尾**：用充满希望或热血的情绪收尾，激发转发欲望。
```

### 8.2 社交货币 (Social Currency)
> 适用于：想让读者主动转发分享。

```markdown
为了让这篇文章更容易被转发，请帮我植入“社交货币”元素：
1. **提供谈资**：提供一个反直觉的观点或冷知识，让读者转发后显得很博学。
2. **表达想法**：设计一段替读者说出心声的金句（嘴替），让他们转发以表达态度。
3. **帮助他人**：整理一份超实用的清单或资源，让读者转发是为了帮助朋友（利他心理）。
```
