# 公众号爆款文章全流程自动化SOP (2025进阶版)

> **版本说明**：本SOP由20年资深内容专家优化，旨在将AI从简单的“生成工具”升级为懂人性、懂算法、懂运营的“全能合伙人”。
> **核心原则**：**RTRS** (Role 角色 - Task 任务 - Requirements 要求 - Structure 结构) + **Iterative Feedback** (迭代反馈)

---

## 🚀 核心理念：人机协作的黄金分工
在开始之前，请明确你与AI的关系：
*   **你是总编辑 (Human)**：负责顶层策略、情感把关、事实核查、最终决策。
*   **AI是超级执行者 (AI)**：负责海量创意发散、结构梳理、初稿填充、多维度优化。

---

## 第一阶段：顶层设计与人格锚定 (Strategy)
**目标**：防止AI写出“千篇一律”的流水账，首先要赋予它独特的“灵魂”和“视角”。

### 【提示词模板：账号人格初始化】
```markdown
你现在是我的公众号首席内容官。请先学习我的账号定位，并将其作为后续所有任务的背景知识。

**账号档案**：
1.  **核心领域**：【如：AI工具提效 / 职场个人成长 / 亲子教育】
2.  **目标用户(Persona)**：【如：25-35岁的一线城市职场女性，焦虑于职业发展与家庭平衡，渴望高效解决方案】
3.  **品牌调性(Tone)**：【如：犀利不失温情、逻辑严密的数据派、像闺蜜一样的碎碎念】
4.  **对标账号**：【列举1-2个你喜欢的同类大号，如有】

**请确认**：
如果你已理解以上定位，请回复：“账号定位已锁定，我已准备好以[品牌调性]风格为您服务。”无需输出其他内容。
```

---

## 第二阶段：选题雷达与爆款库 (Topic Mining)
**目标**：从“由于缺乏灵感”转变为“筛选优质灵感”。结合用户痛点与市场热点。

### 【提示词模板：痛点+热点复合选题法】
```markdown
**角色**：资深新媒体选题策划人
**任务**：基于我的账号定位，结合当前【行业/季节/热点】，挖掘5个具有爆款潜质的选题。

**思维模型**：
请运用**“痛点-痒点-爽点”**模型进行发散：
1.  **痛点 (Fear)**：用户恐惧什么？焦虑什么？(如：裁员、孩子不听话、效率低加班)
2.  **痒点 (Desire)**：用户向往什么样的理想状态？(如：副业月入过万、松弛感生活)
3.  **爽点 (Instant Gratification)**：有什么即时可用的“黑科技”或“神操作”？

**要求**：
1.  输出5个选题方向。
2.  每个选题包含：【暂定标题】、【核心冲突/价值点】、【对标的用户情绪】。
3.  **差异化策略**：指出该选题如何写出新意，避免老生常谈。

**输出示例**：
| 序号 | 选题方向 | 核心冲突/价值 | 差异化切入点 |
| :--- | :--- | :--- | :--- |
| 1 | AI取代职场人 | 焦虑 vs 机遇 | 不谈宏观趋势，只讲3个明天就能用的保命技能 |
```

## 第三阶段：标题炼金术 (Title Engineering)
**目标**：利用人性弱点（贪婪、恐惧、好奇、懒惰）提升打开率，但保持体面。

### 【提示词模板：算法友好型标题生成】
```markdown
**角色**：千万级爆文标题专家
**任务**：针对选题《[插入选定选题]》，撰写10个高点击率标题。

**技巧库**：
1.  **数字悬念法**：如“做对这3件事，胜过瞎忙10年”
2.  **强力否定法**：如“别再学Python了，DeepSeek才是未来”
3.  **借势标签法**：如“阿里P8都在用的……”、“清华学霸的……”
4.  **场景代入法**：如“当我在地铁上看到这一幕……”
5.  **情绪宣泄法**：如“终于有人把这件事说清楚了！”

**要求**：
- 标题要包含强烈的**情绪钩子**或**利益承诺**。
- 覆盖不同风格（震惊类、干货类、故事类）。
- **关键**：请标出每个标题预估的“点击率指数(1-5星)”并说明理由。
```

## 第四阶段：逻辑骨架搭建 (Structuring)
**目标**：不仅要有结构，还要有“阅读滑梯”效应，让读者停不下来。

### 【提示词模板：黄金流结构设计】
```markdown
**角色**：逻辑架构师
**任务**：为标题《[选定标题]》设计文章大纲。

**结构选择**：请根据内容属性选择最合适的结构：
*   **干货类**：SCQA架构 (情境 - 冲突 - 问题 - 答案)
*   **观点类**：PREP架构 (观点 - 理由 - 案例 - 重申)
*   **故事类**：英雄之旅简化版 (平凡世界 - 挑战降临 - 导师指引 - 取得胜利)

**详细要求**：
1.  **开头 (黄金3秒)**：设计一个抓人的引入（提问、扎心场景、反直觉数据）。
2.  **正文 (3-4个核心模块)**：
    - 每个模块的小标题必须是“金句型”小标题。
    - 明确每个模块需要的**素材类型**（是需要数据、案例、还是名人名言？）。
3.  **结尾 (峰终定律)**：设计金句升华 + 强力行动号召 (Call to Action)。
```

## 第五阶段：深度撰写与去AI化 (Drafting & Humanizing)
**关键**：分段生成，便于控制质量。增加“感官细节”是去AI味的核心。

### 【提示词模板：分段式沉浸写作】
> *建议分段发送给AI，不要一次生成全文。*

```markdown
**角色**：[你的具体人设，如：毒舌但真诚的职场导师]
**任务**：撰写大纲中的【第一部分：xxxx】。

**写作军规 (去AI化指南)**：
1.  **禁止说教**：不要用“首先、其次、综上所述”。用“想象一下”、“你有没有发现”等口语替代。
2.  **Show, Don't Tell**：不要说“他很生气”，要说“他猛地摔碎了手里的杯子”。
3.  **短句为主**：模拟微信阅读节奏，多用短段落（不超过3行）。
4.  **情感注入**：在这一段中，插入一句我个人的主观感受或吐槽（括号里的内心戏）。
5.  **金句预埋**：在段落结尾，总结一句适合发朋友圈的金句。

**内容素材**：
[在此处粘贴你需要AI扩写的具体案例、数据或粗糙的想法]
```

## 第六阶段：视觉包装与排版 (Visuals & Formatting)
**目标**：提升完读率和分享率。

### 【提示词模板：视觉与摘要生成】
```markdown
**角色**：视觉总监 & 运营经理
**任务**：为这篇文章做最后的包装。

**1. 封面图提示词 (Midjourney/Stable Diffusion)**：
- 描述画面主体、构图方式、光影风格、配色方案（需符合文章情绪）。
- 示例：`Minimalist illustration, flat design, a tired office worker looking at a glowing AI robot hand offering a cup of coffee, warm orange and blue lighting, high quality --ar 16:9`

**2. 摘要 (Summary)**：
- 提取文章精华，写一段不超过60字的导语，放在文章顶部，诱导阅读。

**3. 金句卡片文案**：
- 提炼文中3句最扎心的金句，用于制作分享海报。
```

## 第七阶段：质量质检官 (Quality Control)
**目标**：自己给自己找茬，模拟严苛的主编视角。

### 【提示词模板：模拟主编审稿】
```markdown
**角色**：拥有20年经验的苛刻主编
**任务**：请批判性地审阅以上生成的文章正文。

**审核维度**：
1.  **逻辑漏洞**：论证是否严密？有无自相矛盾？
2.  **废话检测**：哪些段落是车轱辘话？请指出并建议删除。
3.  **AI味检测**：哪些词汇一看就是AI写的（如“不仅仅是...更是...”、频繁的“至关重要”）？请提供修改建议。
4.  **情绪曲线**：文章是否太平淡？哪里需要加强冲突或共鸣？

**输出**：请列出具体的修改清单，不需要重写全文，只需告诉我哪里要改、怎么改。
```

## 附录：从新手到专家的进阶心法

1.  **建立私有知识库**：不要每次都零起步。将你过往的好文章、你喜欢的素材、你的个人故事整理成文档，喂给AI（使用拥有长上下文的模型的“上传文档”功能），让它越来越像你。
2.  **迭代Prompt**：如果某次生成的效果特别好，把那个Prompt保存下来，建立自己的Prompt库。
3.  **人机耦合**：永远不要直接复制粘贴AI的输出。**AI负责60分（及格与效率），你负责40分（灵魂与连接）。**
